# Spam-detector-streamlit
This is a machine learning powered spam detection system using TF-IDF and Logistic Regression, deployed with Streamlit for real time message classification.
It uses TF-IDF vectorization for text processing and Logistic Regression for classification.
A clean Streamlit web interface allows users to test their own messages in real time.
This project uses the email spam dataset available on Kaggle
Dataset source: https://www.kaggle.com/datasets/devahuja2808/email-spam-dataset
